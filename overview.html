Application Overview

As a user, you should expect to have a singular window open which houses the menu Gui. Once a game is started, the menu Gui will remain open but a new window will open using Turtle that you will use to see the game board which includes the gallows. The correclty guessed and incorrectly guessed letters will be displayed on the menu screen window. You will be able to move the window around.

Instructions

The computer will think of a 5 letter word from the text file below for you to guess...

The game will now commence with the computer drawing the gallows.

You can now begin guessing letters to ultimately guess the word the computer has chosen. Typically vowels "a", "e", "I", "o" and "u" are first to be guessed to maximize the chances of guessing a letter that will be in the word. The next most popular letters include "t", "n", "s", "h", "r", "d" and "l". Of course, which letters to guess will also depend on the word at hand and the letter placement of the letters guessed so far.

For every letter you guess that is within the word the computer has chosen, the computer will write the letter on the dash corresponding to where it is located on the word. For every letter you guess that is not within the word the computer has chosen, the computer must draw a body part of the stick figure on the gallows. The choice of how many body parts to include is locked at 6. A head, torso and four limbs.

If you are able to guess the word before all body parts are drawn on the gallows, then you WIN. If Player 2 has exhausted all guesses with the fact that all body parts have been drawn represented by a hangman, then you have lost.

Technical Specifications

Although hangman is a fairly simplistic game, I do take advntage of some 3rd party libraries as well as my own to get the job done.

Libraries

  • Turtle Graphics (8.4.0)
  • Turtle Gfx Shapes Library (1.0.0)
  • PySimpleGUI (4.57.0)

Dependencies

Hangman requires you to install the following packages in order to have full support

  • Turtle Graphics: Gui used for gameboard.
  • My Word List (bundled into application package): Used to choose what words can be used by the computer. Any word list can be used.
  • PySimpleGUI: Gui interface for the menu and partial gameboard.
  • My Turtle Gfx Library for Shapes (bundled into the application package): Used to draw the gallows and limbs.

You can easily install all of these using pip, just run the following command: pip install turtle pysimplegui

Pre-packaged dependencies are not linked.
